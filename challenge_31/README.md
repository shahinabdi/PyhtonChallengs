# Challenge 31 — Study Checklist

To solve this challenge, you should understand:
- Python's import system: packages, `__path__`, `sys.path`
- `importlib.import_module` for dynamic imports
- `pkgutil.iter_modules` for enumerating a package's modules
- `inspect.getmembers`, `issubclass`, `inspect.isabstract`
- `__init_subclass__` and class-definition-time hooks
- Import-time side effects and their "who imports it?" limitation
- Entry points (`importlib.metadata`) as the production mechanism
