# Challenge 06 — Study Checklist

To solve this challenge, you should understand:
- Generator functions and `yield`
- Why iterating a file object is lazy and `readlines()` is not
- The walrus operator `:=` and the "test-and-use" pattern
- `itertools.islice` for taking at most N items lazily
- `with` blocks inside generators and `GeneratorExit` cleanup
- Separating production (generator) from consumption limits (call site)
