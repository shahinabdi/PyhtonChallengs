# Challenge 07 — Study Checklist

To solve this challenge, you should understand:
- Positional-only parameters: the `/` marker (PEP 570)
- Keyword-only parameters: the bare `*` marker (PEP 3102)
- Why adjacent same-typed parameters cause swap bugs
- Sensible defaults and self-documenting call sites
- Frozen dataclasses (`frozen=True`, `slots=True`) for result values
- `TypedDict` as the alternative for dict-shaped results
- Timezone-aware datetimes (`datetime.now(UTC)`)
