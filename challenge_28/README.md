# Challenge 28 — Study Checklist

To solve this challenge, you should understand:
- The Strategy design pattern and its Pythonic (function-based) form
- First-class functions and `Callable[...]` type aliases
- Decorators with arguments (decorator factories)
- Registration-by-decorator and import-time side effects
- The Open/Closed principle
- Writing actionable error messages (list the valid options)
- `raise ... from None` to suppress internal exception context
