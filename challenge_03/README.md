# Challenge 03 — Study Checklist

To solve this challenge, you should understand:
- `match`/`case` statement basics (PEP 634–636)
- Mapping patterns and why extra dict keys still match
- Capture patterns (binding names inside a pattern)
- Class patterns like `int(n)` for combined isinstance + capture
- Guards (`case ... if condition`)
- Literal patterns for `True`/`False`/`None` compare with `is`, not `==`
- The `case _` wildcard fallback
